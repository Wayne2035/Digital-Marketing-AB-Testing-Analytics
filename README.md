Digital Marketing ROI & Funnel Analysis

This repository shows an end-to-end digital marketing analytics project focused on evaluating channel performance using ROI, CPA, and funnel conversion metrics. The project demonstrates practical data analyst skills using Python, SQL, and structured reporting.

Project Objectives
Measure and compare the performance of digital marketing channels using:
  - Return on Investment (ROI)
  - Cost per Acquisition (CPA)
  - Purchase / conversion rates
Analyze the user journey across funnel stages:
  - Landing → Product View → Add to Cart → Checkout → Purchase
- Identify which channels and funnel stages offer the biggest optimization opportunities.
- Present results in a clear, reproducible format suitable for stakeholders and interviewers.


Tools & Technologies
- Python: exploratory analysis and visualization (Jupyter Notebook)
- Jupyter Notebook: `Digital_Marketing_Analysis.ipynb`
- SQL: KPI calculation (ROI, CPA, purchase rate) and funnel aggregation
- Excel: storing and sharing exported funnel results
